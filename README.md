# Дипломная работа по профессии «Системный администратор»  

# ЗАДАЧА  


Ключевая задача — разработать отказоустойчивую инфраструктуру для сайта, включающую мониторинг, сбор логов и резервное копирование основных данных. Инфраструктура должна размещаться в Yandex Cloud и отвечать минимальным стандартам безопасности.

# Технические детали

Развертывание инфраструктуры автоматизировано с помощью Terraform, Ansible и Gitlab.
Все этапы выполняются в одном пайплайне.
Self-hosted Gitlab работает на локальном домашнем сервере. Раннер запускается в докер-контейнере. Несколько подробнее задокументировал git: здесь/https://github.com/SergeyM90/Diplom/blob/main/Details.md
